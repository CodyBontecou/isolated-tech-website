## Overview
A public `/stats` page showing aggregate metrics. Transparency builds trust with indie software buyers.

## Inspiration
- Plausible's public stats
- Nomad List's open metrics
- Indie Hackers revenue dashboards
- Baremetrics open startups

## Goals
- Build trust through transparency
- Marketing asset (show traction)
- Interesting for indie dev community
- No sensitive data exposed

## Metrics to Display

### Aggregate Stats (Safe to Share)
| Metric | Source |
|--------|--------|
| Total Apps | `COUNT(*) FROM apps WHERE is_published` |
| Total Downloads | `COUNT(*) FROM downloads` |
| Total Reviews | `COUNT(*) FROM reviews + app_store_reviews` |
| Average Rating | `AVG(rating)` across all reviews |
| Countries Reached | `COUNT(DISTINCT country) FROM downloads` |
| Feedback Items | `COUNT(*) FROM feature_requests` |

### Time-Series (Optional, More Complex)
- Downloads per week (last 12 weeks)
- New users per week
- Reviews per week

### What NOT to Show
- Revenue (unless you want to be fully open)
- Individual user data
- Specific app breakdowns (unless desired)

## Implementation

### Database Queries
```typescript
// lib/stats.ts
export async function getPublicStats(env: Env) {
  const [apps, downloads, reviews, countries, feedback] = await env.DB.batch([
    env.DB.prepare("SELECT COUNT(*) as count FROM apps WHERE is_published = 1"),
    env.DB.prepare("SELECT COUNT(*) as count FROM downloads"),
    env.DB.prepare(`
      SELECT COUNT(*) as count, AVG(rating) as avg_rating FROM (
        SELECT rating FROM reviews WHERE is_approved = 1
        UNION ALL SELECT rating FROM app_store_reviews
      )
    `),
    env.DB.prepare("SELECT COUNT(DISTINCT country) as count FROM downloads WHERE country IS NOT NULL"),
    env.DB.prepare("SELECT COUNT(*) as count FROM feature_requests"),
  ]);
  
  return { apps, downloads, reviews, countries, feedback };
}
```

### Caching Strategy
Stats don't need real-time updates:
- Cache in KV for 1 hour
- Or compute via scheduled worker, store results

```typescript
// Check KV cache first
const cached = await env.KV.get("public-stats", "json");
if (cached) return cached;

// Compute fresh
const stats = await computeStats(env);
await env.KV.put("public-stats", JSON.stringify(stats), { expirationTtl: 3600 });
return stats;
```
