## Tasks
1. Add Sentry CLI to GitHub Actions workflow
2. Upload source maps after build
3. Associate releases with commits
4. Verify stack traces are readable in Sentry

## GitHub Actions Addition
```yaml
- name: Upload Source Maps to Sentry
  run: |
    npx @sentry/cli sourcemaps inject ./dist
    npx @sentry/cli sourcemaps upload ./dist
  env:
    SENTRY_AUTH_TOKEN: ${{ secrets.SENTRY_AUTH_TOKEN }}
    SENTRY_ORG: isolated-tech
    SENTRY_PROJECT: website
```
