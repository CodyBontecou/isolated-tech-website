## Current State (app/page.tsx)
1. Get featured app
2. Get all published apps
3. Get review stats for all apps
4. Get latest release dates

## Target
Single query using CTEs:

```sql
WITH app_stats AS (
  SELECT 
    app_id,
    AVG(CAST(rating AS REAL)) as avg_rating,
    COUNT(*) as review_count
  FROM (
    SELECT app_id, rating FROM reviews WHERE is_approved = 1
    UNION ALL
    SELECT app_id, rating FROM app_store_reviews
  )
  GROUP BY app_id
),
latest_releases AS (
  SELECT app_id, MAX(released_at) as latest_release_at
  FROM app_updates
  GROUP BY app_id
)
SELECT 
  a.*,
  s.avg_rating,
  s.review_count,
  r.latest_release_at
FROM apps a
LEFT JOIN app_stats s ON a.id = s.app_id
LEFT JOIN latest_releases r ON a.id = r.app_id
WHERE a.is_published = 1
ORDER BY a.is_featured DESC, a.featured_order ASC, a.created_at DESC
```
