## External services that need user action

### Stripe
- [ ] Create Stripe account (if not done)
- [ ] Get API keys (publishable + secret)
- [ ] Set up webhook endpoint (after deployment)

### AWS SES
- [ ] Verify domain `isolated.tech` for sending
- [ ] Create IAM user with SES permissions
- [ ] Get access key ID + secret

### OAuth Providers
- [ ] **GitHub**: Create OAuth App at https://github.com/settings/developers
  - Callback URL: `https://isolated.tech/auth/github/callback`
- [ ] **Google**: Create OAuth credentials in Google Cloud Console
  - Callback URL: `https://isolated.tech/auth/google/callback`
- [ ] **Apple**: Create Sign in with Apple service ID in Apple Developer
  - Callback URL: `https://isolated.tech/auth/apple/callback`

### Notes
- GitHub OAuth is easiest to set up first for testing
- Apple requires paid Apple Developer account
- Google requires verified OAuth consent screen for production
