## Context
App media (screenshots, icons) management routes have no dedicated tests.

## Routes to test
- `POST /api/admin/apps/[appId]/icon` - Upload app icon
  - Admin auth required
  - File validation (format, size)
  - R2 upload
  - DB update

- `POST /api/admin/apps/[appId]/media` - Add screenshot
  - Admin auth required
  - File validation
  - R2 upload
  - Order assignment

- `DELETE /api/admin/apps/[appId]/media/[mediaId]` - Delete media
  - Admin auth required
  - R2 deletion
  - DB cleanup

- `PUT /api/admin/apps/[appId]/media/reorder` - Reorder media
  - Admin auth required
  - Validate order array
  - Update display_order
